IoT-Based Energy & Flow Monitoring with AWS

This project enables real-time remote monitoring of energy and fluid consumption using IoT and AWS services. It collects data from flow meters and energy meters via RS485, processes it in AWS IoT Core, and visualizes it using AWS IoT SiteWise.

Key Features:

  • IoT Integration: Uses RS485-based flow meters and energy meters for data acquisition.
  • AWS IoT Core: Securely ingests and processes data via MQTT.
  • AWS IoT SiteWise: Transforms raw data into actionable insights.
  • Data Storage in Amazon S3: Stores historical data for analytics.
  • Automated Alarms: AWS IoT Events detects anomalies and triggers alerts.
  • Email Notifications: Alerts are sent using AWS Lambda and Amazon SES.
  • Scalability: Cloud-based processing ensures efficiency and reliability.

Future Work:

  • Predictive Analytics with ML: Forecast energy and fluid consumption trends.
  • Two-Way Pub/Sub Communication: Implement remote control and device adjustments.

College Library Management System

The College Library Management System is a full-stack application aimed at streamlining library operations within an educational institution.

Key Features:

  • Student and Book Management: Unique IDs for students and books. Comprehensive inventory categorized by curriculum (e.g., Computer Science, AI/ML, Mechanical Engineering, Electrical Engineering, programming languages, cloud, DevOps).
  • Issuance and Return Tracking: Manages the process of students borrowing and returning books.
  • Automated Notifications: Sends email reminders to students a day before their book return due date.
  • Fine Calculation: Imposes charges for late returns.
  • Block Mechanism: Prevents students with overdue books or unrecorded returns from issuing new books.
  • Database: SQL database for students, books, current loans, and historical records.
  • Technology Stack: Backend with Node.js and Express; Frontend with React. Includes cron jobs for reminders.
